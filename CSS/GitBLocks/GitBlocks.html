Git init

Git init is the act of registering this current code base.

Git add

Git add is how we stage the intergration of our current code base into our current list of snapshots.

Git status

Git status allows us to see the changes we are going to make.

Git commit

Git commit finally we can commit those changes we added to our last snapshot giving us a new snapshot of our cod with whatever new changes are made.

Git remote add

Git remote add is used to set up an online repository where we can place all of our current code. This is where the outside world can see our code base.
